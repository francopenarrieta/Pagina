Tema 1: Operadores en Programación

Operadores aritméticos, comparacionales y lógicos

Objetivos del tema

  • Comprender qué son los operadores y cuál es su función dentro de un programa.
  • Identificar y utilizar operadores aritméticos para resolver cálculos.
  • Aplicar operadores comparacionales para evaluar relaciones entre valores.
  • Utilizar operadores lógicos para combinar condiciones.
  • Resolver ejercicios integrando operadores en expresiones y decisiones.

1. ¿Qué es un operador?

Un operador es un símbolo que indica al lenguaje de programación que debe realizar una acción sobre uno o más valores. A esos valores se los denomina operandos.

Ejemplo: en la expresión 5 + 3, los números 5 y 3 son los operandos, mientras que el símbolo + es el operador.

Aritméticos

+ − × ÷ %

Permiten realizar cálculos matemáticos.

Comparacionales

== != > <

Permiten comparar valores y obtener verdadero o falso.

Lógicos

&& || !

Permiten combinar o negar condiciones.

2. Operadores aritméticos

Se utilizan para realizar operaciones matemáticas básicas dentro de un programa.

Operador Nombre Ejemplo Resultado
+Suma5 + 38
-Resta10 - 46
*Multiplicación4 * 28
/División10 / 25
%Módulo o residuo10 % 31
**Potencia2 ** 38
Importante: el operador módulo % devuelve el residuo de una división. Por ejemplo, 10 % 3 da como resultado 1.

3. Operadores comparacionales

Comparan dos valores y devuelven un resultado booleano: true (verdadero) o false (falso).

Operador Significado Ejemplo Resultado
==Igual que5 == 5true
!=Diferente de5 != 3true
>Mayor que7 > 5true
<Menor que3 < 6true
>=Mayor o igual que5 >= 5true
<=Menor o igual que4 <= 6true
En JavaScript también existe el operador ===, que compara tanto el valor como el tipo de dato.

4. Operadores lógicos

Permiten combinar varias condiciones o invertir su resultado.

Operador Nombre Ejemplo Descripción
&& AND (Y) edad >= 18 && tienePermiso Las dos condiciones deben ser verdaderas.
|| OR (O) esDocente || esDirector Al menos una condición debe ser verdadera.
! NOT (NO) !activo Invierte el valor lógico de una condición.

5. Ejemplo integrador

El siguiente programa determina si un estudiante aprueba tomando en cuenta su nota y su asistencia: 

let nota = 82;
let asistencia = 90;

if (nota >= 70 && asistencia >= 80) {
  console.log("Aprobado");
} else {
  console.log("Reprobado");
}

En este ejemplo:

  • >= compara la nota y la asistencia.
  • && exige que ambas condiciones se cumplan.
  • El resultado permite tomar una decisión dentro del programa.

6. Errores comunes

  • Confundir = con == o ===.
  • No respetar el orden de las operaciones matemáticas.
  • Usar && cuando en realidad se necesita ||.
  • Olvidar los paréntesis en expresiones complejas.
  • Comparar valores de distinto tipo sin verificar su contenido.

7. Actividades propuestas

  1. Resolver diez operaciones aritméticas.
  2. Predecir el resultado de diez comparaciones.
  3. Determinar si diferentes expresiones lógicas son verdaderas o falsas.
  4. Crear un programa que determine si una persona puede ingresar a un evento según su edad y si tiene entrada.
  5. Crear un programa que indique si un estudiante aprueba según su nota y asistencia.

Reto final

Diseña una expresión que permita verificar si un estudiante:

  • tiene una nota mayor o igual a 70,
  • una asistencia mayor o igual a 80,
  • y además entregó su proyecto final.
