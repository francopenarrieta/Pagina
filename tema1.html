¿Qué es un operador?

Un operador es un símbolo que permite realizar una acción entre uno o más valores. Según la acción que realiza, puede ser aritmético, comparacional o lógico.

Aritméticos

+ − × ÷ %

Sirven para realizar cálculos.

Comparacionales

= ≠ > <

Sirven para comparar cantidades.

Lógicos

Y O NO

Sirven para relacionar condiciones.

1. Operadores aritméticos

Permiten realizar operaciones matemáticas con números.

OperadorNombreEjemploResultado
+Suma8 + 412
−Resta15 − 69
×Multiplicación7 × 321
÷División20 ÷ 45
%Residuo17 % 52
Orden de resolución: primero paréntesis, luego multiplicaciones y divisiones, y finalmente sumas y restas.

2. Operadores comparacionales

Permiten determinar la relación existente entre dos valores. El resultado siempre será verdadero o falso.

OperadorSignificadoEjemploResultado
=Igual que5 = 5Verdadero
≠Diferente de8 ≠ 3Verdadero
>Mayor que9 > 4Verdadero
<Menor que2 < 7Verdadero
≥Mayor o igual que6 ≥ 6Verdadero
≤Menor o igual que3 ≤ 5Verdadero

3. Operadores lógicos

Permiten unir o negar condiciones. También producen como resultado verdadero o falso.

OperadorSignificadoEjemplo cotidiano
YLas dos condiciones deben cumplirse.Puede ingresar si tiene entrada Y documento.
OAl menos una condición debe cumplirse.Puede pagar con efectivo O tarjeta.
NONiega o invierte una condición.NO está lloviendo.
